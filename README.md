## Description
A pre-existing backend providing REST API endpoints was used as the foundation for this project. Based on this API, I built the frontend using Vue 3 + TS + Tailwindcss.<br>

As a result the Upload App is a secure and modern web application that allows authenticated users to upload, store, manage, and share files.
